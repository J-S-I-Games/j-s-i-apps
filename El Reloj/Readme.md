¿Para qué sirve?
El Reloj es una aplicación de seguimiento de tiempo que permite registrar cuánto se dedica a cada actividad durante el día.
Está pensada para ayudarte a tomar conciencia de cómo distribuís tu tiempo y detectar hábitos o rutinas.
No es una agenda ni un simple anotador: mide tiempo real por actividad.

¿Cómo se usa?
Crear o seleccionar una actividad (por ejemplo: Trabajo, Estudio, Juego).
Iniciar la actividad para que el contador comience a registrar el tiempo.
Finalizar o cambiar de actividad cuando sea necesario.
Cerrar el día, la semana o el mes para ver el resumen del tiempo acumulado.
La aplicación organiza automáticamente el tiempo registrado y permite visualizar cuánto se dedicó a cada actividad en distintos períodos
